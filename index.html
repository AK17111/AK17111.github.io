RISC-V Multicore Pipelined CPU

Feb 2025 – May 2025
Purdue University (ECE 437) · SystemVerilog, QuestaSim, Cyclone IV FPGA
  • Built an RV32I core in stages: single-cycle → 5-stage pipeline (full forwarding, load-use hazard detection, BTB/BPT branch prediction) → pipeline with a 2-way set-associative write-back data cache and direct-mapped instruction cache.
  • Extended to a dual-core multicore system sharing RAM over an arbitrated system bus, implementing an MSI snooping coherence protocol plus LR/SC atomics for lock-free synchronization.
  • Verified every module (hazard unit, forwarding unit, caches, bus/coherence FSM) with standalone SystemVerilog testbenches, plus a full-system testbench checked against a cycle-accurate golden model.
  • Benchmarked with merge sort across memory latencies 0–10 cycles: caching alone delivered a ~24× speedup over the uncached pipeline, and the coherent dual-thread multicore added another 1.42× on top of the single-threaded multicore baseline. Synthesized at 50.6 MHz on Cyclone IV (8,622 registers / 19,142 combinational elements).
